# Frontend


Run: `npm install`

To debug/dev run: `npm start`
To build run: `npm run build; npm serve -d build/`

Example JSON from server:

[
		{
			name: "BTC",
			price: "23000",
		},
		{
			name: "ETH",
			price: "1700",
		},
		{
			name: "MATIC",
			price: "0.9",
		},
		{
			name: "XMR",
			price: "130",
		},
];

User clicks the buy or sell button to predict the price movement.
